# PICYBOO Browser Stub Extension (MV3)

Minimal Manifest V3 extension with a popup that hashes the current page URL.

**Status:** Archived • Demo only • No support  
**Trademark:** PICYBOO — US 97338199 / CA 2173851

## How to load
1. Chrome → Extensions → Developer mode → Load unpacked.
2. Select this folder.
3. Click the PICYBOO icon and press *Verify*.
